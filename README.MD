# TypeScript Project Generator

A CLI tool for generating TypeScript projects with a predefined structure.

## Installation

```bash
npm install -g create-my-ts-app
```

Or use it directly with npx:

```bash
npx create-my-ts-app
```

## Usage

Run the command and follow the interactive prompts:

```bash
create-my-ts-app
```

The tool will ask you for:

- Project name
- Whether to include a utils directory
- Whether to include a lib directory
